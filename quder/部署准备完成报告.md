## 🎯 部署步骤（快速参考）

### 步骤 1：部署后端到 Render

**访问**：https://render.com

1. 登录 wsqwsq2003 GitHub 账号
2. New + → Web Service
3. 选择 `portfolio-website` 仓库
4. 配置：
   - Root Directory: `backend`
   - Build: `mvn clean package -DskipTests`
   - Start: `java -jar target/*.jar`
   - 环境变量: `PORT=8080`, `JAVA_OPTS=-Xmx512m`
   - 健康检查: `/api/portfolio/health`
5. Create Web Service
6. 等待 3-5 分钟
7. **复制后端 URL**

### 步骤 2：更新前端配置

**双击运行**：`配置前端环境.bat`

1. 输入 Render 后端 URL
2. 脚本自动更新 `.env.production`
3. 选择提交到 Git

### 步骤 3：部署前端到 Netlify

**访问**：https://netlify.com

1. 登录 wsqwsq2003 GitHub 账号
2. Add new site → Import an existing project
3. 选择 `portfolio-website` 仓库
4. 自动识别配置（已存在 netlify.toml）
5. Deploy site
6. 等待 1-2 分钟
7. **获得网站 URL**

### 步骤 4：测试

访问 Netlify 提供的网站 URL：
- 前台：`https://your-site-name.netlify.app`
- 管理后台：`https://your-site-name.netlify.app/#/admin`

测试实时同步：
1. 打开两个浏览器窗口
2. 一个访问前台，一个访问管理后台
3. 在管理后台修改内容并保存
4. 前台应该自动更新
